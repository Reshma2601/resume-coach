# AI Resume Coach 🎯

An AI-powered resume analyzer that helps job seekers improve their resumes and generate cover letters.

## 🔗 Live Demo
[View Live App](https://resume-coach-theta.vercel.app/)

## ✨ Features
- AI resume analysis with match score (out of 10)
- Visual score circle with color-coded results
- AI-powered cover letter generator
- Analysis history saved to database
- Toast notifications for better UX

## 🛠️ Tech Stack
- Next.js, TypeScript, Tailwind CSS
- Google Gemini AI API
- Supabase (PostgreSQL)
- Vercel
